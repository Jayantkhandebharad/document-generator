Extracted styles and formatting (from uploaded DOCX template)

Goal: Identify sections in the input text and in the template; align and style output according to the template. Use the exact style name as block_type. Numbering (1., 2., 3. or a., b., etc.) and bold phrasing are taken from the template.

Application rules (match template structure)

- Document title / main heading (court name, case caption, or document title like MOTION TO RESTORE) -> use style: Heading 1
- Section headers / subheadings (e.g. VERIFIED COMPLAINT, MEMORANDUM, FACTS, ARGUMENT, CONCLUSION, RELIEF REQUESTED, party captions) -> use style: Heading 6
- Body paragraphs (narrative, legal argument, facts, verification text, general content) -> use style: Normal
- Numbered or bulleted list items (allegations, motion grounds, relief items, any list where the template uses this style) -> use style: Normal (numbers 1., 2., 3. added automatically when applicable).
- Addresses (TO:, court or party address lines): use the same style as in the template (often Normal; one block per line if the template breaks them).
- Signature block (attorney name, firm, address, phone): use the template style for that block; use block_type signature_line for the underline line.
- Separator lines (dashes/dots ending in X) -> use block_type line and put the line characters in text.
- Signature underlines -> use block_type signature_line (optional label in text).
- Section underlines (solid line under a cause of action or heading) -> use block_type section_underline with empty text.
- Page break -> use block_type page_break with empty text before each new major section (as in the template).
